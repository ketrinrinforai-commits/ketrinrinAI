# ketrinrin AI portfolio

Портфолио AI-креатора Екатерины Сапрыкиной: AI-видео для брендов, событий и личных проектов.

## Что внутри

- `index.html` — статичный сайт-портфолио.
- `assets/thumbnails/` — локальные обложки работ.
- `ketrinrin-ai-portfolio.pdf` — PDF-версия портфолио.
- `portfolio-copy.md` — тексты для профиля, постов и описаний.
- `generate_portfolio_pdf.py` — скрипт для пересборки PDF.

## Как открыть локально

Можно открыть `index.html` в браузере или запустить локальный сервер:

```bash
python3 -m http.server 8787
```

После этого сайт будет доступен по адресу:

```text
http://localhost:8787/index.html
```

## Как опубликовать на GitHub Pages

1. Создать новый репозиторий на GitHub.
2. Загрузить в него файлы из этой папки.
3. В настройках репозитория открыть `Settings -> Pages`.
4. В `Build and deployment` выбрать:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Сохранить настройки.

Через пару минут GitHub выдаст публичную ссылку на сайт.
